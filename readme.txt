A cutting edge search widget with Autocomplete. 

== Description ==

Integrate an amazing **search widget** to your widget's sidebar! Use the widget's built-in autocomplete capability to allow your users to search your site quickly and easily.

Feature List:

1. Built-in Autocomplete shows results while the user types
1. Choose your own default value string
1. Graphically attractive, easily customizable, various styles are available
1. Very simple to setup
1. Works with both "ltr" and "rtl" languages and themes

== Installation ==

1. Download the plugin zip package and extract it.
2. Put the folder named "wd_searchform" under /wp-content/plugins/ directory
3. Goto the plugins page in your wordpress admin panel and click "Activate"
4. Goto your widget area (Appearance -> widgets) and add *WD SearchForm* to your widget panel

== Frequently Asked Questions ==

= How do I cancel the animation? =

Go to *Appearance -> widgets* and click on WD SearchForm. Uncheck *Animate?* checkbox.

= Where can i update the settings? / I can't find the plugin settings =

In order to update the settings of the widget, you'll need to login to your wordpress admin panel and navigate to *Appearance -> Widgets* then choose *WD SearchForm* and drag it to the sidebar that you want to use it on.
